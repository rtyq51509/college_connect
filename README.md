## Setup Requirements
Download the latest version from GitHub Releases: https://github.com/rtyq51509/college_connect/releases/tag/CollegeConnect
Click on the first assets,
Extract the file.

- Basic requirements:
A modern browser (Chrome or Edge),
XAMPP,
MySQL.

- Database setup:
Open phpMyAdmin,
Create a new database,
Name the database: college_connect_db,
Import the database file.

- Project Launch:
Run XAMPP,
Enable Apache and MySQL,
Place the project in the following folder:
htdocs,
Open your browser and type:
http://localhost/college_connect-main/home.html
