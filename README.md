## Setup Requirements
- Download the project from GitHub:
Click on Code,
Select Download ZIP,
Extract the file.

- Basic requirements:
A modern browser (Chrome or Edge),
XAMPP,
MySQL.

- Database setup:
Open phpMyAdmin,
Create a new database,
Name the database: college_connect_db,
Import the database file.

- Project Launch:
Run XAMPP,
Enable Apache and MySQL,
Place the project in the following folder:
htdocs,
Open your browser and type:
http://localhost/college_connect-main/index.html
